Selectors

Types of CSS Selectors

  1. Universal Selector: * { }
  2. Type Selector: div { }
  3. Class Selector: .className { }
  4. ID Selector: #idName { }
  5. Attribute Selector: [type="text"] { }
  6. Child Selector: parent > child { }
  7. Descendant Selector: parent descendant { }
  8. Adjacent Sibling Selector: sibling + child { }
  9. General Sibling Selector: sibling ~ child { }
  10. Descendant Selector: parent > child { }
  11. Pseudo-class Selector: a:hover { }
  12. Pseudo-element Selector: p::first-line { }
